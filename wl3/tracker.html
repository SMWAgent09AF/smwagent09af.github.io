Basic WL3 Rando Tracker

by SoapAgent

This serves as a simple, barebones tracker for use with the Wario Land 3 randomizer by AaronDobbe. The location tracker (main part of the page) is contained in one HTML file, the item tracker (iframe in the corner of the page) is contained in another HTML file located here, and the treasure icons are combined into one image located here, so if you want to use them offline you must download all three files. Note that the item tracker uses JavaScript. Read here for more information on how you can customize the tracker.

Here are the features this page includes:

  1. two checkboxes for every level entry in the contents to indicate accessibility and completion of the level
  2. the locations of levels on the map for map shuffle
  3. 4 checkboxes for every level to indicate collection/absence of each chest
  4. 4 dropdowns for each level to indicate where each key is
  5. a textbox for every level in case you want to note something about level requirements, remind yourself to come back, etc.
  6. a tracker for all the useful items in the game, shown in the order given in the treasure menu (you may click on the text at the top to minimize/expand it)

There are no special tools to assist with logical decisions. Each one of these features requires you to click on (or tab over and activate) an element on the page. Use as many or as few of these features as you like.

Depending on how your browser/HTML renderer handles form elements, your input may or may not disappear if you navigate away from this page or refresh. I advise you to know how this works in advance, to avoid accidental clearing of any important information.
